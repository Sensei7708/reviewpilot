Frequently Asked Questions

Everything you need to know about ReviewPilot.

Does ReviewPilot send my code anywhere? ▼
No. ReviewPilot runs entirely on your machine. It uses Ollama to run AI models locally. Your code never leaves your computer — no API calls, no cloud uploads, no data leaks.
Do I need an API key or subscription? ▼
Not for the free version. You only need Ollama installed locally. The Pro license is a one-time payment of $199 — no monthly fees, no per-seat pricing.
What AI models does it support? ▼
Any Ollama model. Popular choices include CodeLlama, DeepSeek Coder, Llama 3, Mistral, and Phi. You can switch models per review or set a default in config.
Does it work with GitLab or Bitbucket? ▼
Yes. GitHub and GitLab are both supported out of the box. For other platforms or custom workflows, use the `diff` command to pipe any diff file.
What happens when I hit the free limit? ▼
Free users get 5 reviews per day. When you hit the limit, you'll see a prompt to upgrade to Pro ($199 one-time) for unlimited reviews, VS Code annotations, and GitHub Action integration.
Can I use it in CI/CD pipelines? ▼
Yes. Pro users get a drop-in GitHub Action that automatically reviews every PR and posts a summary comment. It integrates with your existing workflow in minutes.
